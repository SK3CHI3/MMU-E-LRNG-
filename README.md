## 🚀 Overview

> **Transforming Education Through Technology**

The MMU Learning Management System (LMS) is a state-of-the-art digital platform that revolutionizes the educational experience at Multimedia University of Kenya. Built with modern web technologies and designed with user experience at its core, this comprehensive system serves as the digital backbone for academic excellence.

### 🎯 Mission
To create an intuitive, powerful, and accessible learning environment that empowers students, educators, and administrators to achieve their full potential in the digital age.

### 🌟 Vision
Becoming the leading educational technology platform that bridges the gap between traditional learning and modern digital innovation.

### 🎯 Installation Steps

<details>
<summary><strong>1️⃣ Clone & Setup</strong></summary>

```bash
# Clone the repository
git clone https://github.com/SK3CHI3/MMU-E-LRNG-.git
cd MMU-E-LRNG-

# Install dependencies
npm install
# or
yarn install
```

</details>

<details>
<summary><strong>2️⃣ Environment Configuration</strong></summary>

```bash
# Copy environment template
cp .env.example .env

# Edit .env with your Supabase credentials
VITE_SUPABASE_URL=https://your-project-ref.supabase.co
VITE_SUPABASE_ANON_KEY=your_actual_anon_key
```

> ⚠️ **Security Note**: Check our [Secure Setup Guide](docs/SECURE_SETUP.md) for important security information

</details>

<details>
<summary><strong>3️⃣ Database Setup</strong></summary>

1. Create a new Supabase project at [supabase.com](https://supabase.com)
2. Go to SQL Editor in your dashboard
3. Run deployment scripts:

```sql
-- 1. Schema deployment
-- Copy and paste: database/deploy.sql

-- 2. Sample data (optional)
-- Copy and paste: database/deploy_sample_data.sql
```

</details>

<details>
<summary><strong>4️⃣ Launch Application</strong></summary>

```bash
# Start development server
npm run dev
# or
yarn dev

# Open browser
# Navigate to: http://localhost:8080
```

</details>

### 🎯 **Key Features Documentation**

<details>
<summary><strong>�📢 Announcement System</strong></summary>

The MMU LMS features a comprehensive announcement system designed for effective communication:

**Features:**
- 🚨 **Priority Levels**: Urgent, High, Normal, Low with color-coded visual hierarchy
- 🌐 **Public/Internal Visibility**: Announcements can be public (shown on landing page) or internal only
- 🎯 **Targeted Messaging**: Target specific audiences (all users, students, lecturers, faculty, specific courses)
- 📝 **Rich Content**: Support for external links, expiry dates, and categorization
- ⚡ **Real-time Notifications**: Automatic notifications sent to target users
- 🖱️ **Interactive Display**: Dismissible announcements with read-more functionality

**Visual Design:**
- 🔴 **Urgent**: Red border and high visibility styling
- 🟠 **High Priority**: Orange border for important announcements
- 🟢 **Public**: Green border with globe icon
- 🔵 **Normal/Low**: Blue/gray styling for standard announcements

</details>

<details>
<summary><strong>🗑️ Admin User Deletion System</strong></summary>

The MMU LMS includes a comprehensive user deletion system for administrators:

**Features:**
- 🗑️ **Complete User Deletion**: Permanently remove users and all associated data
- 🔄 **Soft Deletion (Deactivation)**: Temporarily disable users without data loss
- 📦 **Bulk Operations**: Process multiple users simultaneously
- 🔍 **Impact Analysis**: Preview what data will be affected before deletion
- 🛡️ **Safety Measures**: Self-protection and confirmation dialogs
- 📋 **Audit Trail**: Complete logging of all admin actions

**Safety Features:**
- 🚫 **Self-Protection**: Admins cannot delete their own accounts
- ⚠️ **Confirmation Dialogs**: Multiple confirmation steps prevent accidental deletions
- 👁️ **Impact Preview**: Shows exactly what data will be deleted
- ⌨️ **Type-to-Confirm**: Requires typing "DELETE" for permanent deletions
- 📝 **Audit Logging**: All deletion actions are logged for compliance

For detailed information, see [Admin User Deletion Documentation](docs/ADMIN_USER_DELETION.md).

</details>

<details>
<summary><strong>📊 Analytics Dashboard</strong></summary>

Real-time analytics provide insights into academic performance and system usage:

**Lecturer Analytics:**
- 📈 **Course Performance**: Student enrollment trends and grade distributions
- 🎯 **Engagement Metrics**: Assignment completion rates and student participation
- 📊 **Interactive Charts**: Line charts, pie charts, and bar graphs for data visualization

**Dean Analytics:**
- 🏢 **Faculty Overview**: Complete faculty statistics and performance metrics
- 📊 **Department Distribution**: Student and lecturer distribution across departments
- 📈 **Growth Tracking**: Enrollment trends and quarterly growth analysis

</details>
