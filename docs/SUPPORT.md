## ❓ Frequently Asked Questions

<details>
<summary><strong>🔐 Login & Authentication</strong></summary>

**Q: I can't log in to my account**
- Verify your email address is correct
- Check if Caps Lock is on
- Try resetting your password
- Contact admin if you're a new user

**Q: I forgot my password**
- Click "Forgot Password" on the login page
- Check your email for reset instructions
- Contact admin if you don't receive the email

**Q: My account is locked**
- Wait 15 minutes and try again
- Contact admin if the issue persists

</details>

<details>
<summary><strong>👨‍🎓 Student Issues</strong></summary>

**Q: I can't see my courses**
- Check if you're enrolled in the courses
- Verify the semester/academic year
- Contact your lecturer or admin

**Q: Assignment submission failed**
- Check file size (max 10MB)
- Verify file format is supported
- Try refreshing the page and submitting again

**Q: Grades are not showing**
- Grades may not be published yet
- Contact your lecturer for grade status
- Check the announcements for updates

</details>

<details>
<summary><strong>👨‍🏫 Lecturer Issues</strong></summary>

**Q: Can't create assignments**
- Verify you have lecturer permissions
- Check if the course is active
- Contact admin if permissions are missing

**Q: Student list is empty**
- Check if students are enrolled
- Verify the correct semester is selected
- Contact admin for enrollment issues

**Q: Analytics not loading**
- Refresh the page
- Check your internet connection
- Contact support if the issue persists

</details>

<details>
<summary><strong>👨‍💼 Admin Issues</strong></summary>

**Q: Can't delete users**
- Verify you have admin permissions
- Check if the user has dependencies
- Review the deletion warnings carefully

**Q: System settings not saving**
- Check your internet connection
- Verify you have admin permissions
- Try refreshing and saving again

**Q: Analytics dashboard errors**
- Check database connectivity
- Verify data integrity
- Contact technical support

</details>

## 🛠️ Troubleshooting Steps

### 🔄 **General Issues**

1. **Refresh the page** - Many issues are resolved with a simple refresh
2. **Clear browser cache** - Old cached data can cause problems
3. **Try a different browser** - Browser compatibility issues
4. **Check internet connection** - Ensure stable connectivity
5. **Disable browser extensions** - Ad blockers can interfere

### 📱 **Mobile Issues**

1. **Update your browser** - Use the latest version
2. **Clear mobile cache** - Free up storage space
3. **Try desktop version** - Some features work better on desktop
4. **Check mobile data** - Ensure sufficient data allowance

### 🔒 **Security Issues**

1. **Change password immediately** - If you suspect compromise
2. **Log out all sessions** - Use the logout everywhere option
3. **Contact admin** - Report security concerns immediately
4. **Enable 2FA** - When available, use two-factor authentication
